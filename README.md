# Portfolio Landing Page

This is a clean and modern portfolio landing page for Juan Manuel Canovas, built with HTML, CSS, and JavaScript. It features bilingual support (English and Spanish) and is designed to work on GitHub Pages.

## Features
- Clean, modern design with green and black colors
- Bilingual (English/Spanish) with a language switch button
- Sections: About Me, Skills, Experience, Education
- Startup reference at the bottom
- Responsive design

## Setup for GitHub Pages
1. Create a new repository on GitHub (e.g., `username.github.io` for a user site, or any name for a project site).
2. Push these files to the repository.
3. Go to the repository settings.
4. Scroll down to "Pages" section.
5. Under "Source", select "Deploy from a branch".
6. Select the main branch and the root folder (or `/docs` if you put files there).
7. Save, and your site will be live at `https://username.github.io/repository-name/` (or `https://username.github.io/` for user site).

## Local Development
Open `index.html` in your browser to view the site locally.
